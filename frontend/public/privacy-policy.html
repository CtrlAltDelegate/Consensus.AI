2.1 Service Provision

  • Generate AI consensus reports using OpenAI, Anthropic, Google, and Cohere APIs

2.2 Platform Improvement

  • Monitor system performance and identify technical issues

3. Information Sharing and Third Parties

3.1 AI Service Providers

Your content is processed by our AI partners to generate consensus reports:

  • OpenAI: GPT-4 models for analysis and consensus generation
  • Anthropic: Claude models for peer review and analysis

  • Cohere: Command models for final arbitration

These providers process your content solely to provide our services and are bound by their respective privacy policies and our data processing agreements.

3.2 Other Service Providers

  • Stripe: Payment processing (they handle all payment data securely)
  • MongoDB Atlas: Secure cloud database hosting
  • Netlify/Railway: Application hosting and delivery
  • Email Service: Transactional emails and notifications

4. Data Security

We implement industry-standard security measures:

  • Encryption: All data transmitted using HTTPS/TLS encryption
  • Authentication: Secure JWT-based user authentication
  • Access Control: Role-based access with admin privileges
  • Monitoring: Real-time error and performance monitoring
  • Backups: Regular automated backups with encryption
  • Infrastructure: Secure cloud hosting with enterprise-grade security
